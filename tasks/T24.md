# T24: HD Wallet — Frontend TypeScript Types and Utility Functions

**Status**: todo
**Layer**: frontend
**Assignee**: developer
**Depends on**: T23

## Context

This task updates the frontend TypeScript type definitions and utility functions to support HD wallets. It adds `DerivedAddressResponse` to `api.ts`, extends `WalletResponse` with HD fields, and adds `detectBtcInputType` to `validation.ts` and `formatWalletAddress` logic to `format.ts`. These shared utilities must be in place before any UI component can use them.

- [ ] ST4: Update `formatWalletAddress(address, walletType)` (or add if absent) in `frontend/src/utils/format.ts` to truncate HD wallet keys to "first 10 + ... + last 6" and individual addresses to "first 8 + ... + last 6" per §4.10.a
